# MindOS

MindOS is a clean, offline first personal workspace inspired by Notion, built as a single HTML file. It combines a dashboard, daily planner, second brain, decision matrix, and journal into one fast, privacy friendly system that runs entirely in your browser using localStorage.

## Features
- Dashboard with quick links, tasks, and scratchpad  
- Daily planner with time blocking  
- Second Brain for notes and ideas with tags  
- Decision matrix for weighted decision making  
- Journal with mood tracking  
- Fully offline and private  
- No backend, no accounts, no dependencies  

## Usage
1. Download the HTML file  
2. Open it in any modern browser  
3. Start using MindOS instantly  

All data is stored locally in your browser using localStorage.
